**CASE-001 Deep Dive — RCPT**

Full-stack DFIR investigation platform using the Etherscan API. Detects fund splitting, rapid chaining (layering), velocity anomalies, and high-value transfers through a modular rule-based engine. Each wallet receives a weighted risk score (0–100) with interactive graph and timeline visualizations for analyst-driven workflows.

`React` `TypeScript` `TailwindCSS` `Cytoscape.js` `Node.js` `Express` `Etherscan API`
