## Projects

### 🔗 [Ransomware Crypto Payment Tracker (RCPT)](https://ransomware-crypto-payment-tracker.vercel.app/)
> Blockchain forensics dashboard to detect ransomware transaction patterns across Ethereum wallets

Full-stack DFIR investigation platform using the Etherscan API. Detects fund splitting, rapid chaining (layering), velocity anomalies, and high-value transfers through a modular rule-based engine. Each wallet gets a weighted risk score (0–100) with interactive graph and timeline visualizations.

`React` `TypeScript` `TailwindCSS` `Cytoscape.js` `Node.js` `Express` `Etherscan API`
